# Fuentes Recoleta

## Instrucciones para agregar la fuente Recoleta

1. Descarga la fuente Recoleta desde: https://dl.dafont.com/dl/?f=recoleta
2. Descomprime el archivo ZIP descargado
3. Copia los archivos `.otf` o `.woff` de Recoleta a esta carpeta (`src/assets/fonts/`)

### Archivos necesarios:
- `Recoleta-RegularDEMO.otf` (o `.woff`)
- `Recoleta-BoldDEMO.otf` (o `.woff`) - opcional pero recomendado
- `Recoleta-SemiBoldDEMO.otf` (o `.woff`) - opcional pero recomendado

### Nota:
Si los archivos tienen nombres diferentes, actualiza las rutas en `src/index.css` en la secci√≥n `@font-face`.
