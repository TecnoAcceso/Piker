# Fuentes Recoleta

## Instrucciones para agregar la fuente Recoleta

1. Descarga la fuente Recoleta desde: https://dl.dafont.com/dl/?f=recoleta
2. Descomprime el archivo ZIP descargado
3. Copia los archivos `.otf` de Recoleta a esta carpeta (`public/fonts/`)

### Archivos necesarios:
- `Recoleta-RegularDEMO.otf` (requerido)
- `Recoleta-BoldDEMO.otf` (opcional pero recomendado)
- `Recoleta-SemiBoldDEMO.otf` (opcional pero recomendado)

### Nota:
- Los archivos deben tener exactamente estos nombres para que funcionen automáticamente
- Si los archivos tienen nombres diferentes, actualiza las rutas en `src/index.css` en la sección `@font-face`
- Una vez agregados los archivos, reinicia el servidor de desarrollo
